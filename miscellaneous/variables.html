src/app/services/auth.guard.ts

authGuard
Type : CanActivateFn
Default value : (route, state) => { // Inyecta el servicio de autenticación. const authService = inject(AuthService); // Inyecta el router const router = inject(Router); // Si el usuario no esta autorizdo navega al login. if (authService.isAuthorized() === false) { router.navigate(["/login"]) return false; } return authService.isAuthorized(); // Retorna el estado de la autorización del usuario. }

Función para proteger las rutas de usuarios no autenticados. Si no se esta autorzado se redirige al login.
